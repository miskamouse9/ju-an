# Anniversary Page — Mai andd Martin

A small, sweet GitHub Pages site to celebrate our anniversary.  
Copy this repository (Use this template) and edit `docs/index.html` and the assets in `docs/assets/` to personalize.

## How to use
1. Click **Use this template** on GitHub to create your own repository.  
2. Replace `docs/assets/photo.jpg` with your photo(s).  
3. Edit `docs/index.html` to change messages, dates, and song links.  
4. Enable GitHub Pages (Settings → Pages → Source: `main` branch / `docs` folder) — the site will be published at `https://<your-username>.github.io/<repo-name>/`.

## Files
- `docs/index.html` — main webpage (mobile friendly).
- `docs/styles.css` — styles.
- `docs/confetti.js` — lightweight confetti on load.
- `card/printable-card.html` — printable anniversary card (A4/US Letter).
